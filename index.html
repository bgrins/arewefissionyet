Are We Fission Yet?

What is Fission?

Fission is the Firefox's project to implement process isolation per-site (eTLD+1). In a Fission-world, a webpage serving multiple cross-site iframes will be rendered over multiple processes, while multiple tabs or iframes from the same site will share the same process.

You can learn more about Fission on the the wiki or on mana. The Fission team can be reached at #fission on slack or #domfission on IRC.

Milestone 4.1

Fission's current milestone 4.1 aims to get all relevant mochitests to successfully pass in a Fission world. They are currently being run as a tier2 job on mozilla-central and can be seen in the M-fis suite.

More details of the failing tests can be found in this doc and tips to get started on fixing these tests can be found on the wiki.
