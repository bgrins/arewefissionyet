Are We Fission Yet?

What is Fission?

Fission is the Firefox's project to implement process isolation per-site (eTLD+1). In a Fission-world, a webpage serving multiple cross-site iframes will be rendered over multiple processes, while multiple tabs or iframes from the same site will share the same process.

You can learn more about Fission on the the wiki or on mana.

Milestone 4

Fission's current milestone M4 aims to get all Mochitests to successfully pass in a Fission world and is targeted for completion by 31 October 2019. They are currently being run as a tier2 job on mozilla-central and can be seen as M-fis suite.

More details of the failing mochitests can be found in this doc and tips to get started on fixing these tests can be found on the wiki.
